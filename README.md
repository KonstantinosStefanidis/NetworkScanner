A TCP SYN Scanner impemented in python using scapy.
Designed to explore low-level packet crafting and TCP handshake behavior.

This tool uses TCP SYN packets to scan ports of a targeted IP.
Its used in a CLI interface and it can handle specific ports,
or a range of ports.

This tool requires scapy and npcap to work.
